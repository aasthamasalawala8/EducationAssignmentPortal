# 🎓 Education Assignment Portal

## 📌 Description
A web-based platform to manage assignments between students and teachers.

## 🚀 Features
- User Authentication (Login/Register)
- Assignment Upload & Submission
- Student Dashboard
- Teacher Dashboard

## 🛠 Tech Stack
- .NET
- Blazor
- SQL Server
- HTML/CSS

## ▶️ How to Run
1. Clone the repository
2. Open in Visual Studio
3. Configure database
4. Run the project

## 💡 Purpose
Developed as a final year project to simplify assignment management.
